# 🎯 Dork dasar cari file JS

```
site:target.com ext:js
```

---

# 🔑 1. Cari API key di JS

```
site:target.com ext:js intext:"api_key"
site:target.com ext:js intext:"apikey"
site:target.com ext:js intext:"API_KEY"
site:target.com ext:js intext:"secret"
site:target.com ext:js intext:"token"
```

---

# 🔥 2. Cari credential / auth

```
site:target.com ext:js intext:"authorization"
site:target.com ext:js intext:"bearer"
site:target.com ext:js intext:"auth"
```

---

# 💀 3. Cari config sensitif

```
site:target.com ext:js intext:"config"
site:target.com ext:js intext:"env"
site:target.com ext:js intext:"BASE_URL"
site:target.com ext:js intext:"endpoint"
```

---

# 🧠 4. Cari endpoint tersembunyi

```
site:target.com ext:js intext:"/api/"
site:target.com ext:js intext:"/v1/"
site:target.com ext:js intext:"/admin"
site:target.com ext:js intext:"/internal"
```

---

# ⚡ 5. Cari Firebase / third-party key (sering bocor)

```
site:target.com ext:js intext:"firebase"
site:target.com ext:js intext:"firebaseConfig"
site:target.com ext:js intext:"google_api_key"
```

---

# 🚨 6. Advanced combo (lebih tajam)

```
site:target.com ext:js"api_key"
site:target.com ext:js"access_token"
site:target.com ext:js"client_secret"
site:target.com ext:js"Authorization"
```

---

# 🔥 7. Cari JS dari CDN / static

---
```
site:target.com inurl:static ext:js
site:target.com inurl:assets ext:js
site:target.com inurl:js ext:js
```
